# 📊 Myntra Sales Analysis Dashboard (Power BI)

## 📌 Overview
This project presents a comprehensive and interactive sales analysis dashboard built using Power BI.  
The dashboard provides insights into Myntra’s sales performance, customer behavior, product distribution, and order trends across different regions.

It is designed with multiple pages to enable detailed exploration of business data and support data-driven decision-making.

## 🎯 Objectives
- Analyze overall sales and order performance
- Understand customer demographics and behavior
- Identify top-performing products, brands, and categories
- Track monthly sales and order trends
- Provide region-wise insights using map visualization

## 📊 Dashboard Pages

### 🏠 Home Page
- Key KPIs such as Average Age, Total Cities, Total Brands, Categories
- Average Price, Ratings, and Discount Percentage
- Navigation panel for seamless page switching

### 📈 Sales Page
- Product-wise order distribution
- Brand-wise performance analysis
- Category-wise sales breakdown
- Monthly order trends
- Geographic distribution using map visualization

### 📦 Order Page
- Order analysis based on product, brand, and category
- State-wise performance insights
- Monthly order trend comparison
- Sub-category level order distribution

### 📋 Overview Page
- Detailed tabular view of sales across different states
- Brand-level performance comparison
- Comprehensive summary for business analysis

## 📊 Key Insights
- High concentration of orders in top categories like Footwear and Topwear
- Strong brand performance from leading brands like H&M, Puma, and Adidas
- Seasonal variation observed in monthly orders
- Significant contribution from major cities and states

## 📈 Features
- Interactive multi-page dashboard
- KPI cards for quick insights
- Drill-down and filtering capabilities
- Slicers for dynamic analysis
- Map visualization for geographic insights
- Clean and user-friendly UI design

## 🛠 Tools & Technologies
- Microsoft Power BI  
- Power Query (Data Cleaning)  
- DAX (Data Analysis Expressions)  
- Data Modeling

3. Navigate through different pages using the menu  
4. Apply filters and slicers to explore insights
